## Resources
* [Examples][sdk-examples] - A quick, examples to help get you started

## Requirements
* PHP 7.2 and up

## Installation
Via Composer
``` bash
$ composer require gtmc/gtmc-sdk-php
```

## Testing
``` bash
$ vendor/bin/phpunit
```
